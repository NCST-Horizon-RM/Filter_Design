### 滤波器设计

- 八阶巴特沃斯低通滤波器
- 一阶串联型
- matlab直接生成相应系数

| 特性 | 效果                       |
| ---- | -------------------------- |
| 类型 | IIR 巴特沃斯               |
| 通带 | 低通                       |
| 阶数 | 8                          |
| 结构 | 一阶串联                   |
| 参数 | wp=10hz,wst=40hz,ast=-40db |



### 直流分量滤波

### 正弦滤波
选取`signalGenerate`文件产生正弦噪声波。

```c
Frequency_sinNoise_wave(1024, 1.0f, 1000, 100, 50.0f);
```

| 功能     | 参数              |
| -------- | ----------------- |
| 信号振幅 | 1024/4096         |
| 信号频率 | 1hz               |
| 采样点数 | 1000              |
| 采样频率 | 1000hz（0.997ms） |
| 噪声振幅 | 100               |
| 噪声频率 | 50hz              |

![滤除40hz以上噪声，有一定相移](.\img\4.png)

滤波效果良好，**50hz**噪声信号几乎完全被滤除。

经过测试，若噪声波在**20hz**，则无法滤波，滤出的波形几乎和原波形一致，符合设计。
