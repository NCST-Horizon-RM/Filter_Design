## 滤波器设计

> stm32 HAL 库下，实现多种滤波器的整合对比。
> 这是我上完数字信号处理后的一些想法，感觉理论可以同实践结合起来。本质上也是发现没有合适的教程入门滤波器，书本过于强调原理，但仿真终究是仿真。因此我开这样一个坑，帮助想要学习滤波器的同学。
> 希望大家要多多究其原理，这样才能学到更多的知识。
